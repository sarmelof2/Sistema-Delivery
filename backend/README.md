# Backend - Sarmelo Delivery

## Rodar
```bash
cd backend
npm install
npm start
```
## Rotas principais
- POST /auth/register
- POST /auth/login
- GET /menu
- GET /viacep/:cep
- POST /frete
- POST /pedidos (cliente)
- GET /pedidos/meus (cliente)
- GET /pedidos (restaurante)
- POST /pedidos/:id/avancar (restaurante)
